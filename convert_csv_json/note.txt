Overview

The objective of the project is to develop solutions bases on the design provided. In this case, the source data was obtained in the form of CSV files from a MySQL DB.
To improve the effiency of our data enineering pipelines, we need to convert these CSV file into JSON files, since JSON is better to use in downstream application than CSV files. The scope of this project involves converting CSV into JSON files.

Technologies:
+) Programing Language - Python
+) Pandas - For Converting CSV to Dataframe and then Dataframe into JSON
+) glob - Get file names
+) os - Get Runtime Arguments and create folders
